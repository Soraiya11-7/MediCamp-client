# Medical Camp Management System (MCMS)
![Banner Image](./public/image.png)
## üìä  Project Overview

The **Medical Camp Management System (MCMS)** is a **MERN stack** web application designed to help organizers and participants efficiently manage and coordinate medical camps. The system allows **organizers** to create, manage, and track camp registrations, while **participants** can browse, register, and provide feedback. 

It includes **secure JWT authentication**, **payment integration**, **dashboard features**, and **an intuitive user experience**, and is **fully responsive** to ensure accessibility across all devices.

## ‚ú® Features
- **User Authentication** ‚Äì Secure login & registration using Firebase and JWT.  
- **Camp Registration** ‚Äì Users can register for medical camps with payment integration.  
- **Feedback System** ‚Äì Participants can leave reviews and ratings for camps.  
- **Search & Filters** ‚Äì Advanced search, sorting, and filtering for easy camp discovery.  
- **Pagination Support** ‚Äì Camps and participants are displayed with paginated views.  
- **Interactive UI** ‚Äì Fully responsive design built with Tailwind CSS.  
- **Admin/Organizer Dashboard** ‚Äì Organizers can manage camps, payments, and users.  
- **Participant Dashboard** ‚Äì Attendees can track registrations, payments, and feedback.  
- **Payment Integration** ‚Äì Secure online payments via Stripe.  
- **Notifications** ‚Äì Real-time alerts via React Toastify & SweetAlert2.  
- **Data Visualization** ‚Äì Dashboards with analytics and charts (Recharts).

## üõ† Installation

### Prerequisites
Before installing and running the project, ensure you have the following installed:
- **Node.js** (v16 or later) ‚Äì [Download](https://nodejs.org/)
- **NPM** or **Yarn** ‚Äì Comes with Node.js installation
- **MongoDB Database** 
- **Firebase Account** ‚Äì For authentication and storage
- **Stripe Account** ‚Äì For payment processing


### Steps
1. **Clone the repository**

```sh
# Clone the repository
git clone https://github.com/Soraiya11-7/MediCamp-client.git

# Navigate to the project directory
cd MediCamp-client
```
2. **Install dependencies**

```sh
npm install
```
3. **Set up environment variables** (see `.env.local.example` below)

4. **Run the development server**

```sh
npm run dev
```

## ‚öôÔ∏è Configuration (.env.local)

üìå **Create a `.env.local` file** in the root of the project and add the following:

```env
# Firebase Configuration
VITE_apiKey=your_firebase_api_key
VITE_authDomain=your_firebase_auth_domain
VITE_projectId=your_firebase_project_id
VITE_storageBucket=your_firebase_storage_bucket
VITE_messagingSenderId=your_firebase_messaging_sender_id
VITE_appId=your_firebase_app_id

# Image Hosting API Key
VITE_IMAGE_HOSTING_KEY=your_image_hosting_key

# Stripe Payment Gateway
VITE_Payment_Gateway_PK=your_stripe_payment_public_key
```
üîπ Replace `your_value_here` with your actual credentials.

üö® Important: Never expose your .env.local file in public repositories. Use .gitignore to keep it secure.

## üìä Admin/Organizer Dashboard 
The **Organizer Dashboard** provides a structured interface for managing medical camps. It includes:

- **Organizer Profile** ‚Äì Edit profile details like name, image, and contact info.  
- **Add A Camp** ‚Äì Create camps with essential details, ensuring proper validation.  
- **Manage Camps** ‚Äì View, update, or delete created camps in a structured table.  
- **Manage Registered Camps** ‚Äì Track participant registrations, payment status, and cancellations  
- **Manage Registered Camps** ‚Äì Track participant registrations, payment status, cancellations, and confirm enrollments.


---

## üë• Participant Dashboard 
The **Participant Dashboard** offers a personalized experience for camp attendees:

- **Analytics** ‚Äì Visual charts displaying registered camp statistics.  
- **Participant Profile** ‚Äì Update personal information easily.  
- **Registered Camps** ‚Äì View enrolled camps, payment status, feedback, and cancellations.  
- **Payment History** ‚Äì Track transaction details for past and current camp payments.
